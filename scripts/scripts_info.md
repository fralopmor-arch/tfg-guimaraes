# Scripts Folder Quick Guide

This page briefly explains the purpose of each Python file in scripts and scripts/guimaraes.

## scripts

- `manage_curve_outputs.py`
  - Compatibility entrypoint.
  - Delegates to `curves/manage_curve_outputs.py`.

- `plot_curve.py`
  - Compatibility entrypoint.
  - Delegates to `curves/plot_curve.py`.

## scripts/curves

- `curves/manage_curve_outputs.py`
  - Archives `*_curves.csv` from `outputs/curves` into timestamped zip files.
  - Writes a JSON manifest and removes archived CSV files.

- `curves/plot_curve.py`
  - Reads one curve CSV and generates PNG plots.
  - Exports torque-vs-slip and current-vs-slip images.

## scripts/guimaraes

- `guimaraes/catalog_schema.py`
  - CSV schema parsing and validation.
  - Defines `MotorRecord` and grouping/coherence constraints.

- `guimaraes/mode_config.py`
  - Execution mode definitions and size/batch configuration.
  - Resolves validated settings for smoke, fast, intermediate, and full.

- `guimaraes/build_partitions.py`
  - Builds deterministic per-group partition CSV batches.
  - Supports manifest-based partition reuse when inputs are unchanged.

- `guimaraes/physical_guards.py`
  - Applies physical-consistency guards to motor records.
  - Auto-corrects safe issues and rejects invalid rows.

- `guimaraes/guimaraes_model.py`
  - Core equivalent-circuit modeling and predictions.
  - Contains deterministic and legacy estimators plus performance hooks.

- `guimaraes/validate_guimaraes.py`
  - Runs validation against catalog data and computes report metrics.
  - Produces deterministic or deterministic-vs-legacy comparison reports with timing metadata.

- `guimaraes/run_validation_modes.py`
  - Orchestrates mode-based execution end to end.
  - Handles partitioning, per-batch validation, aggregation, runtime controls, and report writing.

## Note

`__pycache__/` folders are generated by Python and are not source files.
